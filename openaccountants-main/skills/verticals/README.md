## How to use

1. Identify the row that matches your business in the table above.
2. Upload that vertical file **plus** your country / state package.
3. Tell the AI which vertical you're in: "I'm a freelance developer in California."

For multi-sided businesses (e.g., e-commerce + content creator), load multiple verticals — they overlay rather than conflict.
