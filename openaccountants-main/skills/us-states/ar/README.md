# Arkansas — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | Arkansas |
| Code | US-AR |
| Income tax | Yes |
| Sales tax | Yes |
| Franchise / gross receipts tax | No (for sole proprietors) |
| Tax authority | [Arkansas Department of Finance and Administration (DFA)](https://www.dfa.arkansas.gov) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder

| File | What it covers |
|---|---|
| `ar-income-tax.md` | Individual income tax for self-employed / sole proprietors (Form AR1000F) |
| `ar-sales-tax.md` | State and local sales & use tax compliance |

## What's NOT covered
- Payroll / employer withholding taxes
- Property tax (administered at county level)
- Corporate income tax (separate rates for corporations)
- Estate / inheritance tax (Arkansas has none)
- Excise taxes (fuel, tobacco, alcohol)
- Special industry taxes (timber severance, etc.)

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
