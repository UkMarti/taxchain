|---|---|

## What's NOT covered
- Corporate income tax
- Net worth tax
- Partnership/S-corp pass-through returns
- Withholding tax
- Motor fuel tax
- Property tax
- Estate/inheritance tax (Georgia has neither)
- Part-year / nonresident allocation details

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
