# Oklahoma — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | Oklahoma |
| Code | US-OK |
| Income tax | Yes |
| Sales tax | Yes (4.5% state rate + local) |
| Franchise / gross receipts tax | No (for sole proprietors) |
| Tax authority | [Oklahoma Tax Commission (OTC)](https://oklahoma.gov/tax.html) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder

| File | What it covers |
|---|---|
| `ok-income-tax.md` | Individual income tax for self-employed / sole proprietors (Form 511) |
| `ok-sales-tax.md` | State and local sales & use tax compliance |

## What's NOT covered
- Payroll / employer withholding taxes
- Property tax (administered at county level)
- Corporate income tax
- Estate / inheritance tax (Oklahoma has none)
- Oil and gas gross production tax
- Franchise tax (applies to corporations, not sole proprietors)
- Tribal-specific tax issues

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
