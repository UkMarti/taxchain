# Hawaii — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | Hawaii |
| Code | US-HI |
| Income tax | Yes |
| Sales tax | Yes — General Excise Tax (GET, 4%–4.5%) |
| Franchise / gross receipts tax | No |
| Tax authority | [Hawaii Department of Taxation](https://tax.hawaii.gov/) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder
| File | What it covers |
|---|---|
| `hi-income-tax.md` | Individual income tax (Form N-11) — graduated 1.4%–11%, self-employed focus |
| `hi-sales-tax.md` | General Excise Tax (GET) obligations for businesses |

## What's NOT covered
- Corporate income tax (Form N-30)
- Transient accommodations tax (TAT)
- Conveyance tax
- Estate tax
- Use tax (separate from GET)
- Partnership/S-corp returns
- Part-year / nonresident returns (Form N-15) in detail
- County surcharge computation details

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
