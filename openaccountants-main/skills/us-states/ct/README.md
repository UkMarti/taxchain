# Connecticut — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | Connecticut |
| Code | US-CT |
| Income tax | Yes |
| Sales tax | Yes |
| Franchise / gross receipts tax | No (for sole proprietors; Business Entity Tax applies to certain entities) |
| Tax authority | [Connecticut Department of Revenue Services (DRS)](https://portal.ct.gov/DRS) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder

| File | What it covers |
|---|---|
| `ct-income-tax.md` | Individual income tax for self-employed / sole proprietors (Form CT-1040) |
| `ct-sales-tax.md` | State sales & use tax compliance |

## What's NOT covered
- Payroll / employer withholding taxes
- Property tax (administered at municipal level — significant in CT)
- Corporate business tax
- Estate / gift tax (Connecticut has an estate tax)
- Pass-through entity tax (PE Tax) elections beyond basic credit
- Unincorporated business tax (CT does not have one)
- Conveyance tax (real estate transfers)

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
