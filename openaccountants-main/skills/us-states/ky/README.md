## What's NOT covered
- Corporate income tax (Form 720)
- Limited Liability Entity Tax — LLET (Form 725)
- Partnership and S-corp pass-through returns (Form 765, 720S)
- Part-year and nonresident returns (Form 740-NP)
- Property tax
- Local occupational license taxes (Louisville Metro, Lexington, etc.)
- Amended returns (Form 740-X)
- Estate/inheritance tax
