# Wyoming — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | Wyoming |
| Code | US-WY |
| Income tax | No |
| Sales tax | Yes (4% state + up to 2% local) |
| Franchise / gross receipts tax | No |
| Tax authority | [Wyoming Dept. of Revenue (DOR)](https://revenue.wyo.gov) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder

| File | What it covers |
|---|---|
| `wy-sales-tax.md` | State and local sales & use tax compliance |

## What's NOT covered
- Property tax (administered at county level)
- Use tax on out-of-state purchases
- Lodging tax
- Mineral severance and production taxes
- Estate / inheritance tax (Wyoming has none)
- Payroll / employer taxes

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
