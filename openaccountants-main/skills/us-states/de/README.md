# Delaware — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | Delaware |
| Code | US-DE |
| Income tax | Yes |
| Sales tax | No |
| Franchise / gross receipts tax | Yes — Gross Receipts Tax (0.0945%–1.9914%) |
| Tax authority | [Delaware Division of Revenue](https://revenue.delaware.gov/) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder
| File | What it covers |
|---|---|
| `de-income-tax.md` | Individual income tax (Form PIT-RES) — graduated 0%–6.6%, self-employed focus |
| `de-gross-receipts-tax.md` | Gross Receipts Tax — rates by business activity, exclusions, filing frequency |

## What's NOT covered
- Corporate income tax (Form 1100)
- Corporate franchise tax (annual report fee)
- Withholding tax
- Real property transfer tax
- Estate tax
- Partnership returns
- Part-year / nonresident returns (Form PIT-NON) in detail
- Wilmington city wage tax (earned income tax)

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
