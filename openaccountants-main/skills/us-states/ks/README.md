## Files in this folder
| File | What it covers |
|---|---|
| `ks-income-tax.md` | Kansas individual income tax (two-bracket progressive: 5.20%–5.58%) for self-employed / sole proprietors. Form K-40. |
| `ks-sales-tax.md` | Kansas sales and use tax — 6.5% state rate plus local taxes. |

## What's NOT covered
- Corporate income tax (Form K-120)
- Partnership and S-corp pass-through returns (Form K-120S)
- Part-year and nonresident detailed allocation (Schedule S Part B)
- Property tax
- Privilege tax for financial institutions
- Amended returns (Form K-40, amended checkbox)
- Withholding filings (KW-3, KW-5)

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM.
