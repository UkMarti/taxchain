# New Mexico — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | New Mexico |
| Code | US-NM |
| Income tax | Yes |
| Sales tax | Yes — Gross Receipts Tax (GRT), not a traditional sales tax |
| Franchise / gross receipts tax | Yes — Gross Receipts Tax (GRT) applies to businesses |
| Tax authority | [New Mexico Taxation and Revenue Department](https://www.tax.newmexico.gov) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder

| File | What it covers |
|---|---|
| `nm-income-tax.md` | Individual income tax for self-employed / sole proprietors (Form PIT-1) |
| `nm-sales-tax.md` | Gross Receipts Tax (GRT) — New Mexico's equivalent of sales tax |

## What's NOT covered
- Payroll / employer withholding taxes
- Corporate income tax (CIT — flat 5.9% effective 2025)
- Property tax (administered at county level)
- Estate / inheritance tax (New Mexico has none)
- Oil and gas severance taxes
- Compensating tax (use tax equivalent)
- Cannabis excise tax
- Insurance premium tax

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
