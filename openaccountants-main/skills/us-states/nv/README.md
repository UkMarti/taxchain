# Nevada — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | Nevada |
| Code | US-NV |
| Income tax | No |
| Sales tax | Yes |
| Franchise / gross receipts tax | Yes — Commerce Tax (businesses with >$4M NV gross revenue) |
| Tax authority | [Nevada Department of Taxation](https://tax.nv.gov) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder

| File | What it covers |
|---|---|
| `nv-commerce-tax.md` | Nevada Commerce Tax — gross receipts tax for businesses with >$4M NV revenue |
| `nv-sales-tax.md` | State and local sales & use tax compliance |

## What's NOT covered
- Modified Business Tax (payroll tax — applies to employers)
- Gaming taxes (NRS Chapter 463)
- Mining taxes (Net Proceeds of Minerals)
- Insurance premium tax
- Live entertainment tax
- Property tax (administered at county level)
- Business license fee (annual $500 fee for most entities)
- Estate / inheritance tax (Nevada has none)

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
