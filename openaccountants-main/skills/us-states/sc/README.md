# South Carolina — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | South Carolina |
| Code | US-SC |
| Income tax | Yes |
| Sales tax | Yes (6% state rate + local, up to 9% combined) |
| Franchise / gross receipts tax | No (for sole proprietors) |
| Tax authority | [South Carolina Department of Revenue (SCDOR)](https://dor.sc.gov) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder

| File | What it covers |
|---|---|
| `sc-income-tax.md` | Individual income tax for self-employed / sole proprietors (Form SC1040) |
| `sc-sales-tax.md` | State and local sales & use tax compliance |

## What's NOT covered
- Payroll / employer withholding taxes
- Property tax (administered at county level)
- Corporate income tax
- Estate / inheritance tax (South Carolina has none)
- Accommodations tax (tourism/lodging)
- Admissions tax
- Business license fees (administered at county/municipal level)

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
