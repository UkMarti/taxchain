# Nebraska — OpenAccountants Tax Skills

| Field | Value |
|---|---|
| State | Nebraska |
| Code | US-NE |
| Income tax | Yes |
| Sales tax | Yes |
| Franchise / gross receipts tax | No (for sole proprietors) |
| Tax authority | [Nebraska Department of Revenue](https://revenue.nebraska.gov) |
| Quality tier | Q3 — AI-drafted, not independently verified |

## Files in this folder

| File | What it covers |
|---|---|
| `ne-income-tax.md` | Individual income tax for self-employed / sole proprietors (Form 1040N) |
| `ne-sales-tax.md` | State and local sales & use tax compliance |

## What's NOT covered
- Payroll / employer withholding taxes
- Corporate income tax (Form 1120N)
- Property tax (administered at county level)
- Estate / inheritance tax (Nebraska has an inheritance tax administered at county level)
- Personal property tax on business equipment

## How to use
Upload all `.md` files in this folder PLUS the federal skills from `skills/foundation/` and `skills/federal/` to your LLM. See the main README for details.
