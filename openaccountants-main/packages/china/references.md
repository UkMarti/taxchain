# China — Related Open-Source Projects

OpenAccountants is AGPL-3.0. Both projects below lack explicit licenses — treat as reference-only (cannot incorporate code directly).

## individual-income-tax-law

- Repository: [cn/individual-income-tax-law](https://github.com/cn/individual-income-tax-law)
- License: no explicit license
- Stars: 24
- Language: Chinese
- Scope: Full text of 中华人民共和国个人所得税法 (PRC Individual Income Tax Law) in Chinese. Reference for the 7-bracket progressive system (3%–45%) and special deductions.
- Integration: No explicit license — reference only. Useful as authoritative text source for validating bracket tables, deduction rules, and tax-exempt income categories. Cannot incorporate code directly.

## personal-income-tax-calculator

- Repository: [YutHelloWorld/personal-income-tax-calculator](https://github.com/YutHelloWorld/personal-income-tax-calculator)
- License: no explicit license
- Stars: 126
- Language: JavaScript / Chinese
- Scope: 2019个税计算器 (Personal income tax calculator). Implements China's 2019 cumulative withholding system with the new 专项附加扣除 (special additional deductions).
- Integration: No explicit license — reference only. Useful for understanding cumulative withholding methodology and special additional deduction (专项附加扣除) logic. Cannot incorporate code directly.
